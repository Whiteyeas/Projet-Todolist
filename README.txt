Projet-Todolist

Installation :
Modifiez les paramètres de connexion dans Database.js et node_modules/user.js  
lancez install.sh

Utilisation : 
lancez startup.sh 
OU
lancez la commande node app.js et allez sur localhost:8080 sur un navigateur/explorateur internet

Si vous voulez modifier le port , celui ci ce trouve dans app.js
